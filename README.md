### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/NBG999/Not-a-nitro-gen.git
   ```
2. Install Python packages
   ```sh
   # Windows:
   python -m pip install -r requirements.txt
   
   # Unix:
   pip3 install -r requirements.txt
   
   # Android (With Termux):
   pkg install python
   pip install -r requirements.txt
   ```
3. Run it
   ```sh
   # Windows:
   python main.py
   
   # Unix:
   python3 main.py
   
   # Android (With Termux):
   python3 main.py
   ```
